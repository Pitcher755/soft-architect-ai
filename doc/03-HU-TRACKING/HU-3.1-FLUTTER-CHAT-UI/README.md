# HU-3.1: Flutter Chat UI

**Estado:** ❌ PENDIENTE
**Prioridad:** High
**Estimación:** L (Large)
**Branch:** feature/ui-chat-widget

## Descripción

Como Usuario, quiero una interfaz de chat limpia que renderice Markdown, para leer código cómodamente.

## Criterios de Verificación

- ❌ El chat sigue los colores del `DESIGN_SYSTEM.md` (Dark Mode).
- ❌ Los bloques de código tienen resaltado de sintaxis y botón de copiar.
- ❌ La UI es responsive al redimensionar la ventana.
- ❌ El botón de enviar está deshabilitado si el campo de texto está vacío.

## Tareas Técnicas

- ❌ Implementar `ChatScreen` usando el prompt de Stitch como referencia.
- ❌ Integrar paquete `flutter_markdown` y `flutter_highlighter`.
- ❌ Crear widgets burbuja de chat personalizados.
